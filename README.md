## Dev Note

### *** Use only yarn and not npm 

POC to demonstrate chatGPT using openAI api

To run local client
```bash
npm start
```
To run local sever
```bash
npm run serve:functions
```
To deploy cloud functions to remote server
```bash
npm run deploy:functions
```
